Cara Transfer Data Antar Device
1
Device A (Pengirim): Buka modal Sinkron Kode → tab Kirim, pilih data yang ingin dikirim, klik Generate Kode.
2
Salin kode yang muncul (klik tombol Salin atau klik teks kode lalu Ctrl+A → Ctrl+C).
3
Kirim kode ke Device B via WhatsApp, email, Telegram, Notes, USB, atau cara apapun.
4
Device B (Penerima): Buka modal Sinkron Kode → tab Terima, tempel kode di kolom teks.
5
Pilih mode (Gabung / Tambah / Timpa), lalu klik Import & Sinkronkan. Selesai!
⚠ Catatan Penting
• Kode Sinkron bersifat teks biasa — bisa dikirim lewat media apapun.
• Kode dikompres dengan Base64 — tidak terenkripsi, jangan kirim jika data sensitif.
• Mode Timpa akan menghapus data lokal yang ada. Selalu backup dulu!
• Sinkron bersifat satu arah — dari pengirim ke penerima, bukan dua arah sekaligus.

Panduan Setup Google Sheets
Setelah setup selesai, semua data Wedding Anda tersimpan di Google Sheets Anda sendiri — aman, gratis, dan bisa diakses dari HP maupun laptop.
Langkah 1 — Buat Google Spreadsheet
Buka sheets.google.com → klik + Spreadsheet kosong → beri nama misalnya "maNTapps Wedding DB".
Langkah 2 — Buat Apps Script
Di Spreadsheet → klik menu Ekstensi → Apps Script → hapus semua kode → paste kode di bawah ini:
Langkah 3 — Deploy sebagai Web App
Di Apps Script → klik Deploy → New deployment → pilih tipe Web App
→ Execute as: Me → Who has access: Anyone
→ klik Deploy → salin URL yang muncul.
Langkah 4 — Hubungkan ke Aplikasi
Kembali ke maNTapps Wedding → Pengaturan → Koneksi Google Sheets
→ paste URL di kolom yang tersedia → klik "Hubungkan & Test" → selesai! ✓
